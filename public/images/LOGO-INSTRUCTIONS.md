# Logo Setup Instructions

## Please save the Cowtrition logo image as:

`public/images/logo.png`

The logo you uploaded should be placed at this exact location for the website to display it correctly.

### Current Logo Requirements:

- **File name**: `logo.png`
- **Location**: `public/images/`
- **Format**: PNG (for transparency) or JPG
- **Recommended size**: 400-600px wide for best quality

The logo will appear:

1. At the top of the hero section (home page)
2. Automatically sized responsively for mobile, tablet, and desktop

Once you place the logo file at `public/images/logo.png`, it will automatically display on the website.
